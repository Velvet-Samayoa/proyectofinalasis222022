Ayuda - Pedido


Proceso Pedido

El proceso de pedido tiene como objetivo registrar un nuevo pedido al sistema.



Debemos llenar los campos vendedor y cliente. Para esto nos colocamos en el boton ? y lo presionamos.



Nos abrira un formulario con todos lo vendedores donde debemos presionar la fila que queremos y oprimir el boton insertar



Luego oprimimos el mismo boton ? para traer el dato insertado.



Haremos exactamente lo mismo con cliente



Ahora para llenar la segunda parte del formulario haremos el mismo paso que realizamos con vendedor solo que ahora lo realizaremos con producto



Cuando cargemos el id del producto se llenaran todos los campos



Ahora introduciremos la cantidad a pedir y oprimiremos el boton agreagar donde se agregará a la tabla si es correcto, en el caso que no haya producto en exitencia o la cantidad sea mayor a la exitencia saldra un mensaje que no hay exitencia.



En el caso que si hay existencia se agregará a la tabla



Para eliminar un producto del pedido oprimiremos la fila del producto a eliminar y el boton eliminar



Ya con los datos ingresador a la tabla tenemos dos opciones para realizar cancelar todo lo que hemos realizado anteriormente o insertar el pedido, para esto oprimiremos los botones respectivamente



Si el Pedido se realiza exitosamente se mostrar un mensaje donde se especifica la fecha limite para realizar el pago y el total
